New BTTENDANCE is Coming!

Version 1.0.0 will be released on 25th, Aug. at Google Play store and Apple App Store! This version will change your experience in BTTENDANCE dynamically. Our team updated the interface cataclysmically and made all the features much more powerful.

Make your lecture interactive with BTTENDANCE!

 

Following is the release note.

 

Clicker

 Clicker is the Audience Response System that enables instructors and students to interact dynamically through question-and-polling.

 
 

- Professors are able to select the number of choices for their own clicker question.

- We offers more information about the result of each clicker. You can see how many students chose each choice and who selected which answer in the Clicker information view.

- Making a clicker question and saving it in advance is now available. Professor can use this feature in the Profile view (located at the top of side menu). 

Attendance

Attendance Check System checks your students' attendance accurately and effortlessly in all ranges of class by using Bluetooth and elaborate algorithm.

 
 

- In this version, Professors can select attendance checking method either ‘Check Attendance faster with Bluetooth’ or ‘Check Attendance by calling each student’.

- You can check each student’s attendance status. Also, you can toggle it in the order of Absent -> Present -> Tardy -> Absent by clicking student’s name. 

Notice

Notice makes easy to notify important information to your students.

 
 

- Professors are able to check how many students read the notice so far.

 

Sign Up & Create Course Process

 
 

- Only full name and email address are required to sign up.

- You can either search or register your institution.

- After creating a course, class code (identification code for each course) will be generated automatically. Students may register your course easily by entering the class code.

 

User Interface

 
 

- Side menu has been added.

- You can update your personal information in the Profile view.

- In addition, you can decide whether to turn on/off push notification for each BTTENDANCE action in the Setting view.

- You can give us feedback easily by clicking Feedback button!

 

Guide & Tutorial

 
 

- Guides and Tutorials for each feature are supported.

Version 0.3 release note

For Professor & Students

1. Clicker: We are using “socket.io”, which is the hottest technique these days, for Clicker. Enjoy students’ real-time responses!

2. User Interface renewal

- Course List: BTTENDANCE features (Clicker, Attendance, Notice) are available in the course list view. Buttons for adding supervising courses and attending courses are now integrated with a plus button in the top bar.

- Course Detail: BTTENDANCE features (Clicker, Attendance, Notice) are also available in the course detail view. More information about the courses is offered.

- Feed: Professor can see the rate of each attendance check. Notice with more than one sentence is supported.

3. Bluetooth 3.0/2.0 for attendance check is now supported.

4. More universities such as KAIST, SNU, Yonsei Univ, Korea Univ, Postech have been registered.

For Professor

1. Validation Procedure: The validation procedure to create courses has been simplified from serial validation to email validation.

2. Grades Export: The attendance grade of your students can be exported as an excel file. You can receive the grade via email.

For Student

1. Course Unjoin: Unjoining from your courses is now supported.
